# Dockerized CI/CD Pipeline using Multinode Jenkins

This project contains a simple Docker-based web application to demonstrate a complete CI/CD pipeline using:

- Jenkins Master (Instance 1)
- Jenkins Agent / Deploy Node (Instance 2)
- Docker
- GitHub

## Pipeline Stages
1. Pull code from GitHub  
2. Build Docker image  
3. Push image to Docker Hub  
4. Deploy container on Deploy Node
